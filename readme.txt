 ✈️ Airline Route Analysis using SQL

 📊 Project Overview

This project analyzes airline flight data to understand route-wise traffic, delays, and airline performance.

 🔍 Key Analysis Performed

* Identified busiest routes (origin → destination)
* Analyzed departure and arrival delays
* Evaluated airline performance across routes
* Studied cancellation trends
* Performed city-level traffic analysis

 🛠️ Tools Used

* SQL (MySQL)

📈 Key Insights

* Found high-traffic routes with maximum flight frequency
* Identified routes with highest delays
* Observed travel demand patterns across cities

📂 Dataset

Dataset based on airline flight data (BTS-inspired dataset)
